## Overview

E-SOS is a website designed to raise awareness about gender-based violence and provide support resources for victims of abuse. The platform aims to educate the public, encourage reporting of abuse, and connect individuals with organizations that offer assistance and protection.

## Features

* Home Page with Awareness Campaign Information
* About Us Section
* Information on Gender-Based Violence
* Contact Page
* Support and Help Resources
* Responsive Design for Mobile and Desktop Devices
* Easy Navigation Interface

## Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Development Tools

* Visual Studio Code
* GitHub

## Project Objectives

## Installation

1. Download or clone the project repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in a web browser.
4. Navigate through the website pages using the menu.

## Purpose

This project was developed to support the mission of helping individuals affected by abuse by providing information, resources, and awareness through a modern web platform.
